Cómo usar este formateador de JSON

Pega cualquier cadena JSON en el panel de entrada. Pulsa Formatear para verlo con resaltado de sintaxis y la sangría elegida (2 o 4 espacios). Pulsa Minificar para comprimir los espacios en blanco en payloads de producción. Si el JSON no es válido, el mensaje de error muestra la línea y columna exactas donde falló el análisis para que puedas corregirlo rápidamente.

Al pegar JSON se formatea automáticamente — no hace falta pulsar ningún botón. Pulsa Copiar para copiar la salida limpia al portapapeles. Todo se ejecuta en tu navegador: no se envía ningún dato a ningún servidor, por lo que es seguro con respuestas de API, ficheros de configuración y datos privados.

Preguntas frecuentes

¿Se sube mi JSON a algún sitio?

No. Formatear, validar y minificar ocurre localmente en tu navegador con JavaScript. Nada de lo que pegues se envía a un servidor, lo que lo hace seguro para respuestas de API, ficheros de configuración y otros datos privados.

¿Admite comentarios o comas finales (JSON5)?

No, esta herramienta valida JSON estricto según la especificación oficial. Los comentarios y las comas finales aparecerán como un error de sintaxis con la línea y columna exactas para que puedas corregirlos o eliminarlos.

¿Cuál es la diferencia entre Formatear y Minificar?

Formatear añade sangría y saltos de línea para facilitar la lectura. Minificar elimina todos los espacios en blanco para producir el payload más pequeño posible en producción.

¿Hay un límite de tamaño para el JSON que puedo pegar?

La herramienta en sí no impone un límite estricto, pero los payloads muy grandes (varios MB) pueden ralentizar el formateo, ya que todo se ejecuta en la pestaña de tu navegador en lugar de en un servidor.
