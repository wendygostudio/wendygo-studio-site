Preguntas frecuentes

¿Qué motor de regex usa esta herramienta?

Usa el motor de regex nativo de JavaScript (ECMAScript), el mismo que corre en Chrome, Firefox y Node.js — así que los patrones que pruebes aquí se comportan igual en tu código.

¿Admite flags de regex como g, i, m y s?

Sí, puedes activar los flags habituales para probar coincidencia global, insensibilidad a mayúsculas, modo multilínea y comportamiento dot-all.

¿Puedo probar con texto multilínea?

Sí, pega cualquier bloque de texto en el campo de cadena de prueba — las coincidencias se resaltan en tiempo real mientras editas el patrón o el texto.

¿Se envían mis datos de prueba a algún sitio?

No, la coincidencia se ejecuta íntegramente en tu navegador mediante el RegExp nativo de JavaScript — nada se envía a un servidor, por lo que es seguro probar con logs o datos privados.

Referencia Rápida de Regex

Una referencia condensada de la sintaxis de expresiones regulares más usada. Todos los patrones de este probador siguen el motor de regex ECMAScript de JavaScript.

TokenSignificadoEjemplo
.Cualquier carácter excepto salto de línea (añade el flag s para incluirlos)c.t → "cat", "cut"
\dCualquier dígito 0–9\d\d\d → "404"
\wCarácter de palabra [a-zA-Z0-9_]\w+ → "hello_world"
\sEspacio en blanco (espacio, tabulador, salto de línea)\s+ → huecos entre palabras
\bLímite de palabra\bcat\b coincide con "cat" no con "catch"
^Inicio de cadena (o de línea con el flag m)^\d → línea que empieza con un dígito
$Fin de cadena (o de línea con el flag m)\d$ → línea que termina con un dígito
*0 o más (voraz)ab* → "a", "ab", "abbb"
+1 o más (voraz)ab+ → "ab", "abbb" (no "a")
?0 o 1 (opcional)colou?r → "color" o "colour"
{n,m}Entre n y m veces\d{2,4} → de 2 a 4 dígitos
[abc]Clase de caracteres (a, b, o c)[aeiou] → cualquier vocal
[^abc]Clase negada (cualquiera excepto a, b, c)[^\d] → no dígito
(abc)Grupo de captura — guardado como $1, $2…(\d{4})-(\d{2}) → dos grupos
(?:abc)Grupo sin captura(?:foo|bar)baz
a|bAlternancia (a o b)cat|dog
*? +?Cuantificador perezoso (coincidencia más corta)<.+?> coincide solo con la primera etiqueta

Flags

FlagNombreEfecto
gGlobalEncuentra todas las coincidencias, no solo la primera
