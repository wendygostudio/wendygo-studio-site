  • Limpiar: espacios extra, saltos de línea, trim, quitar HTML, eliminar acentos

  • Extraer: correos, URLs, IPs (v4 y v6), números
  • Desarrollo: Base64, URL, HTML, JSON, escapar regex, slugify, hex
  • Generar: SHA-256, UUID, contraseñas seguras, Unix ↔ fecha
  • Markdown: viñetas, listas numeradas, bloque de código, envolver en 80 cols
  • ✨ Recetas: encadena transformaciones en pipelines reutilizables
  • ✨ Compositor IA: lenguaje natural → receta vía Gemini Nano local
  • Forge Magic: pega JSON/Base64/hex y recibe sugerencias de 1 clic
  • Búsqueda y reemplazo con regex + filtro grep con regex (Pro)
  • Recetas guardadas y composiciones IA ilimitadas (Pro)
  • Generador de tabla Markdown desde columnas (Pro)

Privacidad

Tus datos se quedan contigo

Todas nuestras extensiones están construidas alrededor de una única restricción: nada sale de tu navegador.

🔒
Permisos mínimos
Cada extensión solicita solo lo necesario para funcionar. La validación de pago es un permiso opcional, pedido solo al pulsar Activar.
📵
Sin servidor
Todas las transformaciones y el procesamiento de imágenes se ejecutan completamente en tu pestaña del navegador. No hay backend.
🗑
Sin recopilación de datos
No recopilamos, almacenamos ni transmitimos datos personales ni análisis de uso.
🖼
Las imágenes nunca se suben
FrameForge procesa imágenes localmente. Tus fotos nunca dejan tu máquina.
🔑
BYOK — tu clave, tus datos
ScrubForge utiliza tu propia clave de API de IA. Nunca vemos, proxy ni almacenamos tus consultas.
