Alle Formate, eine Ablagezone

  • Images: HEIC, PNG, JPG, WebP, GIF, BMP, ICO, AVIF, SVG in → PNG, JPG, WebP, ICO out — resize on the fly
  • Audio: MP3, WAV, OGG, FLAC, M4A — or extract audio straight from MP4/WebM videos
  • Documents: images → PDF, PDF → images, PDF → text, Markdown ↔ HTML
  • Data: JSON, JSONL, CSV, YAML, XML, XLSX, Base64 — paste or drop, convert between any
  • Universal drag-and-drop router: drop anything and it lands in the right converter
  • Pro: local OCR (scanned PDF → searchable text via Tesseract, 100% on-device)
  • Pro: unlimited batch conversion, favicon pack generator, download interceptor for .webp/.avif
