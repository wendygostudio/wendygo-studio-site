Datenschutz

Deine Daten bleiben bei dir

Die Kernverarbeitung erfolgt lokal. Daten verlassen deinen Browser nur, wenn du ausdrücklich einen externen Anbieter mit deinem eigenen API-Schlüssel auswählst.

Minimale Berechtigungen
Jede Erweiterung fordert nur das an, was sie zum Funktionieren benötigt. Die Zahlungsvalidierung ist eine optionale Berechtigung, die nur abgefragt wird, wenn Sie auf „Aktivieren“ klicken.
Kein Wendygo-Server
Alle Transformationen und Bildverarbeitungen laufen vollständig in Ihrem Browser-Tab. Es gibt kein Backend.
Keine Datenerfassung
Wir erheben, speichern oder übermitteln keine personenbezogenen Daten oder Nutzungsanalysen.
Bilder werden nie hochgeladen
FrameForge verarbeitet Bilder lokal. Ihre Fotos verlassen nie Ihr Gerät.
BYOK — dein Schlüssel, deine Daten
ScrubForge verwendet Ihren eigenen AI-API-Schlüssel. Wenn Sie sich ausdrücklich an einen externen Anbieter wenden, gehen die Daten gemäß dessen Bedingungen direkt an diesen Anbieter. Wendygo sieht die Anfrage weder, noch leitet sie sie weiter oder speichert sie.
