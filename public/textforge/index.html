Features

Everything you need for text

  • Clean: remove extra spaces, line breaks, HTML, accents
  • Case: UPPERCASE, lowercase, Title Case, Sentence case
  • Sort: A-Z, Z-A, numeric with KB/MB/GB, deduplicate
  • Extract: emails, URLs, IPv4/v6, numbers
  • Dev: Base64, URL encode, HTML, JSON, regex escape, slugify
  • Generate: SHA-256, UUIDs, passwords, Unix date conversion
  • Markdown: bullet list, numbered list, code block, wrap at 80 cols
  • Recipes: chain transformations into reusable workflows
  • Forge Magic: detect JSON, Base64 and hex when you paste
  • Local AI Composer: describe a workflow and let Gemini Nano build the recipe on-device when available
  • Pro Only: regex find & replace, column extraction, word frequency
