Confidentialité

Vos données restent avec vous

Le traitement principal s’effectue localement. Les données ne quittent le navigateur que si vous choisissez explicitement un fournisseur externe avec votre propre clé API.

Autorisations minimales
Chaque extension ne demande que ce dont elle a besoin pour fonctionner. La validation du paiement est une autorisation facultative, demandée uniquement lorsque vous cliquez sur Activer.
Aucun serveur Wendygo
Toutes les transformations et tous les traitements d'image s'exécutent entièrement dans l'onglet de votre navigateur. Il n'y a pas de back-end.
Aucune collecte de données
Nous ne collectons, ne stockons ni ne transmettons aucune donnée personnelle ou analyse d’utilisation.
Images jamais téléversées
FrameForge traite les images localement. Vos photos ne quittent jamais votre machine.
BYOK — votre clé, vos données
ScrubForge utilise votre propre clé API AI. Lorsque vous invoquez explicitement un fournisseur externe, les données sont directement transmises à ce fournisseur selon ses conditions ; Wendygo ne voit jamais, ne transmet pas et ne stocke jamais la requête.
