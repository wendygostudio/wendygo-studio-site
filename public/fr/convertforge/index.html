Chaque format, une zone de dépôt

  • Images : HEIC, PNG, JPG, WebP, GIF, BMP, ICO, AVIF, SVG en entrée → PNG, JPG, WebP, ICO en sortie — redimensionnez à la volée
  • Audio : MP3, WAV, OGG, FLAC, M4A – ou extrayez l'audio directement à partir de vidéos MP4/WebM
  • Documents : images → PDF, PDF → images, PDF → texte, Markdown ↔ HTML
  • Données : JSON, JSONL, CSV, YAML, XML, XLSX, Base64 – coller ou déposer, convertir entre n'importe quel
  • Routeur universel par glisser-déposer : déposez n'importe quoi et il atterrit dans le bon convertisseur
  • Pro : OCR local (PDF numérisé → texte consultable via Tesseract, 100 % sur l'appareil)
  • Pro : conversion par lots illimitée, générateur de packs de favicon, intercepteur de téléchargement pour .webp/.avif
