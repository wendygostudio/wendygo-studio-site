Privacy

Your data stays with you

Core processing runs locally. Data only leaves your browser when you explicitly choose an external provider with your own API key.

Minimal permissions
Each extension requests only what it needs to work. Payment validation is an optional permission, asked only when you click Activate.
No server
All transforms and image processing run entirely in your browser tab. There is no backend.
No data collected
We do not collect, store or transmit any personal data or usage analytics.
Images never uploaded
FrameForge processes images locally. Your photos never leave your machine.
BYOK — your key, your data
ScrubForge uses your own AI API key. When you explicitly invoke an external provider, data goes directly to that provider under its terms; Wendygo never sees, proxies or stores the query.
