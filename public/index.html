T
TextForge
paste · transform · copy

All-in-one text toolkit with 50+ functions to clean, convert, extract and manipulate text. Paste your text, pick a transform and copy the result — find any function instantly with global search.

  • Clean: extra spaces, line breaks, trim, strip HTML, remove accents
  • Case: UPPERCASE, lowercase, Title Case, Sentence case
  • Sort: A→Z, Z→A, numeric (understands KB/MB/GB/TB), deduplicate
  • Extract: emails, URLs, IPs (v4 & v6), numbers
  • Dev: Base64, URL, HTML, JSON, regex escape, slugify, hex
  • Generate: SHA-256, UUID, secure passwords, Unix ↔ date
  • Markdown: bullets, numbered lists, code block, wrap at 80 cols
  • Regex find & replace + regex grep filter (Pro)
  • Column extractor and word frequency analysis (Pro)
  • Markdown table generator from columns (Pro)

FrameForge
resize · adjust · export

An image resizer and thumbnail maker for YouTubers, streamers and content creators. Paste from clipboard, drag in or pick a file — crop to the exact platform size and export. Everything processed in your browser.

  • YouTube thumbnail, Shorts, Reels, Instagram, Twitch, X header, custom
  • Fill / Fit / Stretch with zoom and drag to reposition
  • Rotate 90°, flip horizontal & vertical, circle crop
  • 4 effects: gradient, vignette, cinematic bars, black & white
  • JPEG / PNG / WebP with adjustable quality and live file-size
  • Copy to clipboard, EXIF metadata stripped on export
  • 8 extra effects: warm, cool, dramatic, vintage, glow… (Pro)
  • Text layers and logo/image overlays (Pro)
  • Brightness, contrast & saturation controls (Pro)
  • Batch processing and multi-size ZIP export (Pro)
  • Target size compression — e.g. max 2 MB (Pro)

ScrubForge
sanitize · share · ask

A config sanitizer for sysadmins, network engineers and MSP consultants. Paste your Cisco, FortiGate, MikroTik or Palo Alto config — ScrubForge detects and replaces sensitive data with consistent tokens, then ask AI about it without exposing your infrastructure.

  • 120 patterns across 5 categories: network, secrets, PII, infrastructure, custom
  • 12 vendor profiles: Cisco, FortiGate, MikroTik, Juniper, Palo Alto, Dell…
  • Consistent tokens preserve topology — same IP → same token
  • BYOK — use your own key with Claude, GPT-4o, Gemini, Groq or Mistral
  • Format-aware tokens: SHA-512 stays SHA-512 (Pro)
  • Integrated AI chat with multi-turn conversations (Pro)
  • Custom dictionary for values regex doesn't catch (Pro)
  • All 12 vendor profiles and audit mode (Expert)
  • Batch processing and question obfuscation (Expert)
