Privacy

I tuoi dati restano con te

L’elaborazione principale avviene localmente. I dati lasciano il browser solo se scegli un provider esterno con la tua chiave API.

Autorizzazioni minime
Ogni estensione richiede solo ciò di cui ha bisogno per funzionare. La convalida del pagamento è un'autorizzazione facoltativa, richiesta solo quando si fa clic su Attiva.
Nessun server Wendygo
Tutte le trasformazioni e l'elaborazione delle immagini vengono eseguite interamente nella scheda del browser. Non esiste un back-end.
Nessuna raccolta dati
Non raccogliamo, archiviamo o trasmettiamo alcun dato personale o analisi di utilizzo.
Immagini mai caricate
FrameForge elabora le immagini localmente. Le tue foto non lasciano mai la tua macchina.
BYOK: la tua chiave, i tuoi dati
ScrubForge utilizza la tua chiave API AI. Quando richiedi esplicitamente un fornitore esterno, i dati vanno direttamente a quel fornitore secondo i suoi termini; Wendygo non vede, inoltra o memorizza mai la query.
