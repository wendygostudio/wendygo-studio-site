Pinterest's algorithm favors vertical pins. When you size images for the platform, you are competing with thousands of other pins for space in users' feeds — and the wrong dimensions get cropped or stretched.

The standard Pinterest pin size is **1000×1500 px** (2:3 aspect ratio). Some creators use 1000×1200 px or 1000×1300 px, but the 1000×1500 format displays perfectly on both mobile and desktop feeds with no auto-cropping.

## FAQ

**What is the ideal Pinterest pin size?**
Pinterest recommends 1000×1500 px as the ideal vertical pin ratio (2:3). This format displays optimally on mobile and desktop feeds. You can also use 1000×1200 px (5:6 ratio) or 1000×1300 px — all perform well as long as the width is 1000 px.

**Does FrameForge have a Pinterest preset?**
Yes. FrameForge includes a preset for Pinterest pins at 1000×1500 px. Open the extension, select the Pinterest preset, adjust the crop, and export. No account, no upload, no third-party service.

**Can I resize to other Pinterest dimensions?**
Yes. If you need a different size, FrameForge allows custom dimensions. You can set any width and height you need — for example, 1000×1200 for square pins or 1200×900 for horizontal pins, though vertical format typically performs best on Pinterest.
