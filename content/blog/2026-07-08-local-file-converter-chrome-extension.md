# Local File Converter Chrome Extension: Images, Audio, Documents — No Upload

Most online converters take your file, send it to a server, process it there, and return a result. Your photo, document, or spreadsheet lives briefly on someone else's infrastructure. ConvertForge converts files entirely inside your browser — nothing leaves your machine.

## What ConvertForge Converts

ConvertForge is a universal conversion router with a drag-and-drop interface. Drop a file and it detects the type automatically:

- **Images** — HEIC from iPhones and other image formats, converted locally without a cloud service
- **Audio** — Extract audio from video files or convert between audio formats
- **Documents** — Local OCR using Tesseract, extracting text from PDFs and images on-device
- **Data** — Convert between JSON, CSV, YAML, and XLSX for data pipelines or spreadsheet imports

## How to Convert a File Locally

1. Install ConvertForge from the ConvertForge page — no account, no sign-up
2. Click the ConvertForge icon in your Chrome toolbar to open it as a local panel
3. Drag and drop your file onto the universal router
4. ConvertForge detects the file type and shows available output formats
5. Select the target format and convert — the file downloads directly to your device

All processing uses browser-native APIs: Tesseract for OCR, WebAssembly for audio and image processing.

## Frequently Asked Questions

**Does ConvertForge upload my files to any server?**
No. ConvertForge runs entirely in your browser using browser APIs and WebAssembly. Nothing is uploaded to Wendygo Studio's servers or any third party.

**Does it work offline?**
Yes. Because processing is local, ConvertForge works fully offline once installed — on a plane, with no Wi-Fi, or on a machine with restricted network access.

**Is it free?**
ConvertForge has a free tier with no account required. Visit the ConvertForge page for current pricing.

**How is it different from online converters?**
Online converters upload your file to a remote server and process it there. ConvertForge does all of this in your own browser — your file never leaves your machine.
